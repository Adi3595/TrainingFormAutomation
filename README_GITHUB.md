# Training Management System (GitHub Notes)

This repo is a full-stack app:
- **Frontend**: React (in `frontend/`)
- **Backend**: Node.js/Express (in `backend/`)

## Setup

### Backend
1. `cd backend`
2. `npm install`
3. Create `backend/.env` from `../ENV_EXAMPLE.md`.
4. `npm start`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm start`

## Important
- Do not commit secrets or credentials. See `ENV_EXAMPLE.md`.
- Large/example datasets and uploaded files are ignored by `.gitignore`.
